# Portfolio Optimisation Notebook

This folder contains the main analysis notebook developed collaboratively as part of the ADS2001 portfolio optimisation project.

The notebook implements the complete data science workflow, including:

- Data cleaning and preparation of historical S&P500 stock data
- Construction of an investable stock universe
- Portfolio optimisation across 25+ strategies
- Risk analysis and performance evaluation
- Static and dynamic portfolio allocation methods
- Out-of-sample strategy comparison and statistical testing

The analysis compares classical optimisation, risk-based approaches, tail-risk methods, and machine learning-driven strategies using historical data from 1993–2019.

For an overview of the project findings and recommendations, refer to the main README. For detailed methodology, results, and recommendations, refer to the final report.
