# Final Presentation

**Unit:** ADS2001 – Data Challenges  
**Year:** 2025

## Overview

This folder contains the final project presentation developed for ADS2001.

The presentation was delivered to stakeholders to communicate the project background, data analysis process, modelling approach, key findings, and recommendations.

## Purpose

The presentation aimed to:

- Summarise the problem and project objectives
- Explain the data preparation, exploratory analysis, and modelling approaches
- Present key insights from the analysis
- Provide recommendations and suggestions for future data-driven approaches

The presentation was designed for a non-technical stakeholder audience, focusing on communicating insights and supporting decision-making.
